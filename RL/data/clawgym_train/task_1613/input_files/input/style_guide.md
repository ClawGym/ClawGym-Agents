Audience: small business owners and non-technical readers.

Rewrite requirements:
1) Use a calm, reassuring tone. No blame.
2) Keep each message ≤ 80 words.
3) Avoid acronyms/jargon: RTO, RPO, DNS, NXDOMAIN, drift, pipeline. Use plain alternatives (e.g., recovery time, recovery point, domain name system, name lookup issue).
4) State whether there's impact; include a clear action requested (if any) or reassurance if no action is needed.
5) Prefer short sentences and present tense.
