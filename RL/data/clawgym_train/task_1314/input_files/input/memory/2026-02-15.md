## insight
### Noticed automated backups reduced sync errors
Noticed that adding nightly automated backups (pre-sync) reduced sync error rates from ~4.2% to ~1.1% over two weeks. Hypothesis: fewer partial writes during power interruptions. Appears likely that preflight backups plus atomic state writes increase reliability. Follow-up: test on larger workspaces. Keywords: automation, reliability, sync.
