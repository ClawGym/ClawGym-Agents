# Draft comparative review: metal kit guides

I read two guides and wanted to jot down my thoughts.

Guide A (Pewter Knight Herald) felt very detailed. It talks about washing parts and using CA glue and pinning. It also mentions self-etching primer and waiting 24 hours. This seemed slow but maybe solid. I liked the safety notes about respirators.

Guide B (Universal) is faster and simpler. It suggests plastic cement is okay and says to just use any acrylic primer and start painting in an hour. This would save time. It doesn’t go deep into surface prep. I didn’t see much on metal-specific tips, but it’s easy to follow.

Preliminary thought: For speed painting, the universal guide seems convenient. For durability, the metal workflow could be better, but I haven’t tied this to my adhesion tests yet. I also didn’t weigh different criteria; this is just a rough take. Needs structure and scoring.
