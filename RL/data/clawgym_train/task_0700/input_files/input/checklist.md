# Q2 House Checklist

- Change HVAC air filter (MERV 11) and set next reminder
- Flush water heater tank and test T&P relief valve
- Vacuum refrigerator condenser coils and clean toe grille
- Test all smoke detectors and replace batteries
- Clean gutters and downspouts after spring blooms
- Start irrigation system and check for leaks/spray alignment
- Test sump pump and backup battery
