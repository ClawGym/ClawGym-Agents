# Pipeline Brief

Summarize the sales funnel for the current month:
- Highlight counts by status (new, contacted, qualified, converted, lost).
- Include total leads, converted count and conversion rate.
- Report total and average deal value for converted leads.
- Note active pipeline (new + contacted + qualified).
- Briefly mention lead sources distribution if present.

Structure:
1. One-paragraph overview with the month.
2. Bulleted metrics pulled from the CLI report.
3. A short interpretation (e.g., where leads concentrate and next steps).
4. Links section:
   - Link to the raw CLI output at ../pipeline.txt.
