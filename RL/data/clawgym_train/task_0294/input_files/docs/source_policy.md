# Editorial Source Policy

Our publication values accuracy, fairness, and transparency. Named sources are preferred, and claims should be corroborated where possible.

## Existing Standards
- Fact-checking is mandatory for all investigative pieces.
- Sensitive allegations require editor sign-off and, where needed, legal review.
