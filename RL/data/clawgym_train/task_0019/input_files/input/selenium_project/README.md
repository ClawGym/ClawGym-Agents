# Sample Selenium Project (Demo Only)

This is a minimal Selenium + pytest example used for internal testing demos. It assumes a Selenium hub is reachable at `http://selenium-hub:4444/wd/hub` and a demo app at `http://localhost:8080`.

- Configuration lives in `config/test_config.yaml`.
- Tests are under `tests/`.
- Credentials in this demo are hardcoded for convenience and should be replaced in real pipelines.

Do not use these defaults in production.
