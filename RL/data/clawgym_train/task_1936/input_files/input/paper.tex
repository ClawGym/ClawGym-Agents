\begin{abstract}
We revisit the sonic metaphors in early 20th-century poetry, aligning textual rhythm with the emergent urban sensorium \cite{Eliot1922}.
\end{abstract}

As argued by Smith \cite{Smith2020}, soundscapes structure readerly attention. We extend this reading in light of archival discoveries \cite{MissingKey}.
