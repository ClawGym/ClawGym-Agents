# Brand Voice Guide

Tone:
- Inquisitive, empathetic, and historically grounded.
- Inclusive language; avoid sensationalism and jargon.
- Be precise with dates/terms when used; if uncertain, frame as a question.

Style:
- Clear, short paragraphs.
- Prefer active voice and plain language.
- One thoughtful question per outreach email to invite dialogue.

Email specifics:
- Subject line should start with: "Partnership Idea: " followed by a concise descriptor.
- Close with: "Warmly, [Your Name]" (leave the name placeholder as is).

Positioning:
- Emphasize how food history illuminates societal change.
- Invite collaboration that benefits educators, culinary enthusiasts, and community organizers.
