# Laptop Donation Device Policy (Draft)

Purpose: Establish baseline security expectations for laptops we donate to students in our tutoring program.

Current draft (to be updated):
- Screen should auto-lock after 15 minutes of inactivity.
- Full-disk encryption is recommended (optional) to protect student data.
- Guest account is discouraged but allowed if supervised.

- This draft will be finalized after we validate the automation approach.
