# Methodology Draft

This project examines how post-Civil War industrialization reshaped Southern labor markets and regional capital formation from 1865 to 1900. The empirical strategy triangulates multiple primary source families to infer patterns in wage dynamics, manufacturing output, and capital investment.

## Data Sources
- Government: decennial census manufacturing schedules (selected counties), state labor/statistics reports, and federal administrative records (e.g., Freedmen's Bureau).
- Corporate: annual railroad reports summarizing mileage, payroll, and investment.
- Private: plantation account books documenting wages, inputs, and credit.
- Press: digitized Southern newspapers for job postings and price signals.

## Sampling and Abstraction
I plan to sample representative counties and enterprises where documentation is sufficiently complete to support county-level comparisons over time. Variables will be harmonized where possible (e.g., industry categories, wage units, denominations) and annotated when comparability is ambiguous.

## Transformations
- Standardize monetary units to 1880 dollars using commonly accepted deflators.
- Normalize counts (e.g., establishments per 1,000 workers) for cross-county comparisons.
- Aggregate micro-records to county-year panels when feasible.
