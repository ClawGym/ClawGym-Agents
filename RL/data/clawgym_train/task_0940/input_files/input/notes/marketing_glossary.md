# Marketing and Onboarding Glossary

Progressive profiling
- Progressive profiling: collecting additional user attributes over time rather than all at once.
- Benefits: lower signup friction, better timing, higher data accuracy.
- Related: staged data capture, just-in-time questions.

Progressive disclosure
- Progressive disclosure: revealing controls or fields only when they are relevant to the task.
- Goal: reduce cognitive load while preserving power for advanced users.

Activation
- Activation: the moment a new user experiences core product value.
- Examples: first file uploaded, first project created, first report generated.

Value moment
- Value moment: a concrete success event that signals perceived utility.
- Often used as a trigger for contextual prompts.

Segmentation
- Segmentation: grouping users by attributes or behavior to tailor experiences.
- Input sources: declared profile data, behavioral events, inferred attributes.

Consent
- Consent: a user’s unambiguous indication of agreement for specific processing purposes.
- Must be granular, informed, and easy to withdraw.

Preference center
- Preference center: a user interface to review and manage communication and personalization settings.
- Should include consent scopes, channels, and topics.
