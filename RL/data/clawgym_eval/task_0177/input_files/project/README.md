# Mountain Studio Portfolio (Starter)

A friend shared this to help me put my paintings online. Rough notes:

- Run `npm install` to set things up; extras may be fetched automatically after install.
- Use `sh site/build.sh` to build. If a helper is missing, the script will try to fetch it.
- Temporary passwords were used during the harvest festival; I plan to change them later.

Please don’t run anything on untrusted networks.
