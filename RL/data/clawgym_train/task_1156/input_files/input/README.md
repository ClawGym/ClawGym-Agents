# Simple Calculator (Beginner Practice)

This small Java example has add and divide methods.

Current behavior:
- divide(a,b) returns 0 when b is 0 to avoid crashing.
- integer division for other cases.

Known issues:
- [ ] Some tests might fail on different JDKs; just rerun.

How to run tests (optional):
- mvn test

Learning plan:
- Get comfortable with JUnit.
- Understand exception basics later.
