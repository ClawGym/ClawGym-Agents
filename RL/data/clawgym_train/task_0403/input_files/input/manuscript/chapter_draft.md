# Hippocampal Circuits and Signal Propagation

In the CA-1 region of the hippocampus, pyramidal neurons integrate synaptic input and produce output that propagates along Schaffer collaterals. The astro-glia in the local microcircuit modulate synaptic transmission and potassium buffering, which is essential for maintaining excitability.

The canonical view of spike generation traces back to biophysical models of the membrane, for example [ref:hodgkin_huxley_1952]. For modern computational treatments, see also [ref:dayan_abbot_2001].
