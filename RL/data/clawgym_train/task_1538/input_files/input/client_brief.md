Client Brief — Atlas Notes
Date: 2026-06-30

Overview
Atlas Notes is a lightweight, Obsidian-style note bundle focused on rapid project documentation using wikilinks, callouts, embeds, properties (frontmatter), and footnotes. The initial scope is a minimal, coherent set of interlinked notes for a small team.

Goals
- Atlas Notes MVP goals: deliver a Project Hub, a Design Decisions log, and a Research note that demonstrate Obsidian-flavored features (frontmatter, wikilinks, callouts, embeds, highlights, footnotes).
- Ensure all internal links resolve within the bundle (no external files).
- Keep structure simple and discoverable for first-time contributors.

Scope & Constraints
- Content must be English-only.
- Use Obsidian wikilinks for intra-bundle navigation.
- Avoid images or binary assets in the MVP.
- Keep headings stable to support section links and embeds.

Technical Preferences
- Text-first workflow with Markdown files.
- Optional: Use PostgreSQL in later phases for metadata indexing and search; do not add external dependencies in MVP.

Deliverables
- A working Project Hub that links to Research and Design Decisions, with a small diagram illustrating navigation.
- A Research note importing the agreed “Key Findings” and “References” sections.
- A Design Decisions note with at least one concrete decision and a linkable block ID.

Timeline
- MVP assembly and review by 2026-07-07.
- Iterate after stakeholder feedback.

Success Criteria
- All internal links and embeds resolve correctly.
- MVP demonstrates Obsidian-specific features clearly and consistently.
- Tasks are listed as a checklist with owners and statuses.
