# Iron Meridian — Special Makeup/Prosthetics Protocol v1.0

Scope: This protocol covers facial prosthetics, adhesives, and removal for principal combat days.

General Guidelines:
- Confirm allergy status and document skin sensitivities.
- Favor water-based or skin-safe adhesives when feasible.
- Ensure proper ventilation and cooling during extended takes.

Application:
- Edges must be feathered and sealed before movement sequences.
- Allow adequate set time prior to close-ups.

Removal (General):
1. Use approved removers; avoid aggressive rubbing.
2. Support skin while lifting edges.
3. Cleanse and moisturize post-removal.
