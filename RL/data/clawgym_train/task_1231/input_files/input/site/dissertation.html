Abstract: This dissertation documents the formatting and submission process for a doctoral thesis, including institutional repository deposit, embargo options, and compliance with APA style. It discusses coordinating with the committee and the defense timeline, IRB considerations, and avoiding plagiarism. Methods include LaTeX templates and reproducibility practices for data sharing.

We cover ORCID linking, DOI assignment, metadata requirements, and how open science improves discoverability in the university repository. The methodology section explains version control, quality checks, and how to prepare datasets for review.

Key terms: thesis submission, abstract, methodology, committee, defense, institutional repository, embargo, ORCID, DOI, APA, LaTeX, data sharing, reproducibility, plagiarism, IRB.
