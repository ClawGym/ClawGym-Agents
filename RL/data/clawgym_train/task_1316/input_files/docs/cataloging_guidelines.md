# Digitized Records Cataloging Guidelines v1.0

## Purpose
These guidelines establish baseline practices for cataloging digitized historical records.

## Categories
- Community
- Industry
- Family

## Field Naming
- record_date: string; use ISO-like format (YYYY-MM-DD) when known; year-only is acceptable if that is all that is available.
- place_name: free text; city/town followed by county/state when known.

## Notes
- Use the fewest necessary categories and consult the archivist before adding new ones.
- Maintain consistency in field naming across records.
