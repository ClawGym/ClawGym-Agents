# AI Lab CI/CD Guidelines

This repository currently uses a simple CI workflow for teaching purposes.

## Triggers
- CI runs on any branch push and on pull requests.

## Python versions
- Tests run only on Python 3.10.

## Caching
- No dependency caching is configured.

## Concurrency
- No concurrency policy is set.

## Job timeouts
- No explicit timeout is set.
