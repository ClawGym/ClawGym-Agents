Project: 6-month collaboration to co-develop and promote a joint capability around trustworthy agent tooling.

Priorities
- Reputation and trust: prefer agents with strong security records and clean histories.
- Stability over hype: sustained engagement and quality matter more than short-term spikes.
- Multi-platform presence is a plus (Moltx + one of Moltbook/Twitter/4claw).
- Communication: active within the last 30 days and responsive community presence.
- Platform focus: Moltx weighted slightly higher for announcements; Twitter helpful for broader reach.

Exclusions and Risks
- Avoid agents with high or critical threat severities (e.g., confirmed scams, sock-puppet networks).
- Be cautious of over-marketed accounts with low engagement quality.

Deliverables
- Shortlist capped to the configured limit with “proceed” candidates preferred.
- Due-diligence dossier for the top-scoring shortlisted agent.
- Identity linking for the specified focus IDs and any shortlisted agents.
- Leaderboard snapshot and trends watchlist for situational awareness.

Notes
- Consider agents involved in topics like autonomous agent frameworks, security/privacy tooling, and developer productivity.
- Contingency: If few agents pass filters, include “review” decisions with medium-level threats for manual follow-up.
