# Chapter: Beginnings

I remember the first rehearsal like it was yesterday. The bass rattled the walls and we laughed about how the neighbors would never forgive us. Some of the background on the band's arc is well summarized here: [Type O Negative - Wikipedia](https://en.wikipedia.org/wiki/Type_O_Negative).

For Peter's broader story, this was useful: [Peter Steele - Wikipedia](https://en.wikipedia.org/wiki/Peter_Steele). When we started developing the textures for the mid-90s records, this was a touchstone: [October Rust - Wikipedia](https://en.wikipedia.org/wiki/October_Rust).

For context on how the press heard us at the time, I often check [NPR Music](https://www.npr.org/music/) and the [NPR Music News](https://www.npr.org/sections/music-news/) feed. Some interviews that capture the era's mood show up on [All Songs Considered](https://www.npr.org/sections/allsongs/).

I also keep an eye on archives of fanzines and flyers: [Internet Archive collection](https://archive.org/details/TypeONegative).
