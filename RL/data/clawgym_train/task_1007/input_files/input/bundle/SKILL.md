# Research Insight Assembler

Use this skill to turn fragmented research (notes, links, excerpts) into:
- Executive briefs
- Structured outlines
- Draft deliverables with citations

Guidelines:
- Ask for the audience, purpose, and length.
- Preserve key facts and attribution.
- Prefer clear structure (headings, bullets) and an executive summary.
- Flag any gaps or weak evidence before finalizing.

Inputs can include pasted notes, URLs with quoted excerpts, and short user briefs. Outputs should be concise, structured, and easy to review.
