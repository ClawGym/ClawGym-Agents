# Feishu Doc Skill

This skill teaches agents how to:
- Authenticate to Feishu using app credentials
- Publish messages to groups and DMs
- Handle file uploads and markdown rendering

Configuration tips:
- Keep app secrets in ~/.openclaw/secrets.json
- Test in a sandbox chat first
- Avoid broad scopes unless strictly required
