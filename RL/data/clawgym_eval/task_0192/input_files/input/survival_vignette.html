Cox proportional hazards model

The Cox model estimates hazard ratios without specifying the baseline hazard.


library(survival)
fit <- coxph(Surv(time, status) ~ age + sex, data = lung)
summary(fit)

Assumptions

  • Proportional hazards: the effect of covariates is multiplicative and constant over time.
  • Linearity on the log hazard scale for continuous covariates unless transformed.
  • Independent censoring.
