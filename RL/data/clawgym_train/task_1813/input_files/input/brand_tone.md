Brand Voice & Tagline

Tagline:
Rooted in tradition, crafted for today.

Voice guidelines:
- Warm, respectful, and clear. Avoid slang.
- Use metric units for volumes (ml) and refer to cases succinctly.
- Use the currency code before amounts (e.g., KES 22500).
- Keep subjects and body copy calm and confident; avoid multiple exclamation marks.
- Keep closing friendly and concise.
