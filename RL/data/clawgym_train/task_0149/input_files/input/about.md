# About HelioAnalytics

HelioAnalytics is a clean-energy software company helping operators of distributed solar and storage assets maximize yield while reducing operational burden. We serve:
- Commercial & Industrial (C&I) asset managers
- O&M contractors managing multi-state fleets
- Municipal and small utility-scale programs

## What We Build
- HelioScore 2.0: probabilistic baselines for accurate anomaly detection
- Fleet KPIs with revenue-at-risk rollups
- Workflow integrations (ServiceNow, Zendesk, Slack) and open APIs
