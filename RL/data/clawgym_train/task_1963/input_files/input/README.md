# Sample Project

This is a small, self-contained repository used to exercise read-only filesystem
inventory operations. It includes:

- Python package code under `src/`
- Documentation under `docs/`
- Data files under `data/`
- Notes under `notes/`

Quick start:

1. Inspect the tree
2. Parse the JSON and YAML samples
3. Review the JSON Lines dataset
4. Run the CLI module to print a summary
