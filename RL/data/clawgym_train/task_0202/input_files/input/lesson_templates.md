# Flight Training Syllabus (Template)

Module M1 — Emergency Procedures
- Lesson L1: Engine Failure on Takeoff
  - Scenarios covered: Engine failure on takeoff
  - Planned hours: 2
  - Objectives: Recognize and execute immediate actions.

- Lesson L2: Rapid Decompression
  - Scenarios covered: Rapid decompression
  - Planned hours: 1.5
  - Objectives: Don oxygen, emergency descent.

- Lesson L3: Hydraulic Failure
  - Scenarios covered: Hydraulic failure
  - Planned hours: 1.5
  - Objectives: Manual reversion procedures.

- Lesson L4: Electrical Fire
  - Scenarios covered: Electrical fire
  - Planned hours: 1
  - Objectives: Smoke and fire isolation.

Module M2 — Navigation & Systems
- Lesson L5: Navigation System Outage
  - Scenarios covered: Navigation system outage
  - Planned hours: 2
  - Objectives: Raw data navigation.

- Lesson L6: GPS Deviation and Reversionary Mode
  - Scenarios covered: GPS deviation
  - Planned hours: 1
  - Objectives: Reversionary mode operations.

Module M3 — Landing Techniques
- Lesson L7: Crosswind Landing
  - Scenarios covered: Crosswind landing
  - Planned hours: 1.5
  - Objectives: Control inputs and go-around criteria.

- Lesson L8: Short-Field Landing
  - Scenarios covered: Short-field landing
  - Planned hours: 1
  - Objectives: Performance calculations and technique.
