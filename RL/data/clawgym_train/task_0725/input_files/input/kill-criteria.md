# Project Kill Criteria — Acorn CLI

We will stop investing in Acorn if any of the following conditions are met, and do not improve within the specified timeframe:

- Adoption: Weekly active users (WAU) < 100 by 2026-06-30 and no improving trend for 4 consecutive weeks.
- Quality: More than 3 unresolved P0 issues for > 14 consecutive days.
- Stability: More than 3 hotfix releases in a single calendar month for 2 consecutive months.
- Strategic fit: If Acorn’s roadmap no longer aligns with core user needs identified in quarterly user interviews.

If a kill criterion is met, we will:
1) Freeze new feature work
2) Publish a deprecation plan (60-day window)
3) Archive the repository

Owner: Release Lead
Review cadence: Monthly
