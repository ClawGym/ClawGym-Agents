# Historical Fiction Workshop: Voices of 1773

## Overview
This 60–90 minute session uses in-progress chapters to explore voice, point of view, and the lived texture of a moment in time. Participants will read selected pages, discuss character presence and absence, and propose a micro-scene to bridge a documented gap in the story’s timeline.
