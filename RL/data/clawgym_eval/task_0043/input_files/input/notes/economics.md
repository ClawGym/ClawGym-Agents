Explanation:
In basic portfolio theory, the expected value of a return is $E[R] = \\sum_i p_i r_i$, and the variance is $\\sigma^2 = E[(R - \\mu)^2]$. Probability and statistics help quantify risk from data. A simple model lets us see how changes in probability affect expected value and variance.

```text
Example: with two outcomes p=[0.6, 0.4], r=[0.05, -0.02], E[R] = 0.6*0.05 + 0.4*(-0.02) = 0.022.
```
