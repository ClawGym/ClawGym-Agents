Insignia Brand Color & Stroke Guidelines (v1.0)

Approved color hex codes:
- Navy: #0A1F44
- Gold: #D4AF37
- White: #FFFFFF

Palette rules:
- Use only the approved hex codes above.
- A single insignia must not exceed 3 unique colors.

Stroke rules:
- All strokes must be exactly 1.5px wide.
- Stroke width units must be px.

Scope:
- This audit applies to all SVG files under assets/insignia.
