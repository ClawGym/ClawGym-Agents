# Thank-You Letter Outline

- Title: A short, sincere title.
- Opening: Greeting and personal note in a humble voice.
- Gratitude: Acknowledge community support and car assistance.

- Helpers: Bullet list of top helpers (name — count).
- Community Quote: Include one quote verbatim from community_quotes.md with attribution.
- Closing: Thanks and a forward-looking note.
- Data snapshot: Include script_version and the path to derived/metrics.json.
