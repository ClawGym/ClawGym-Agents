# Intel Notes

- Dolores Park Statue: Busy 18:00-19:30; easier after 19:30.
- Coit Tower: Limited parking; better after 20:00.
- Ferry Building: Crowded on event nights; watch for security.
- AT&T Park Gate: Watch for ballpark events; good turnover when games end.
- Twin Peaks Vista: Windy; visibility good; low foot traffic.
