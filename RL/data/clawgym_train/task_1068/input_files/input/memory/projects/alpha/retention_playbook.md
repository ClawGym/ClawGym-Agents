# Alpha Project Retention Playbook

This playbook documents how to keep new contributors engaged after onboarding and how to reinforce good note-taking and search habits.

## Follow-up Cadence

- Day 1: Welcome message with links to MEMORY.md and search_tips.md
- Day 7: Retention check — confirm they indexed memory and can run hybrid search
- Day 21: Deeper check — assess their ability to find decisions using headers and file paths
- Day 30: Review salience prompts and stale topics they surfaced

## Onboarding Retention

Tie onboarding checklist items to retention milestones:
- Ensure they completed the onboarding checklist from 2026-04-15 daily note.
- Ask them to find that daily note by querying “onboarding checklist and decisions” and observe temporal routing behavior.
- Reinforce adaptive weighting best practices when keyword overlap is low.

## Notes Hygiene

- Keep decisions under a “Decisions” header.
- Add project names to directories and filenames for filepath matching.
- Avoid overly generic language that weakens keyword overlap.

## Metrics

- Search success rate with hybrid approach
- Time to first relevant note
- Number of stale but important topics surfaced via salience scoring
