# Sample Web + API Repository

This is a sample web+API project used for automated quality gate evaluation.

Contents:
- API service (Node/Python hybrid)
- Web frontend (React + TypeScript)
- CI artifacts under `tests/`, `security/`, `perf/`
- Dependency staleness report under `deps/`
- SBOM evidence under `sbom/`
- Operability docs under `ops/`

Use `.defs/quality-gateway-definition.json` for scoring thresholds and output paths.
