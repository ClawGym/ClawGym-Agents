Social Teaser Selection Rules and Tone Guide

Selection Rules:
1. Select exactly 5 quotes from the transcripts. Each quote must be the full "text" field from a single line and must be 200 characters or fewer.
2. The 5 selected quotes must collectively cover at least 3 distinct themes (from: resilience, loss, courage, ethics, fear, hope, aftermath). No more than 2 quotes may share the same theme. Use the line's "tags" to determine themes.
3. Exclude any quote containing these substrings (case-insensitive): "blood", "gore".
4. Set requires_cw = yes if the quote's tags include "loss" or "aftermath"; otherwise set requires_cw = no.
5. Include the source and timestamp for attribution.

Tone Guide:
- Keep captions compassionate and respectful. Avoid sensationalism and graphic detail.
- Emphasize resilience, agency, and care over spectacle.
- When in doubt, prefer language that centers human dignity.
