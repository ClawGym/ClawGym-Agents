Explanation:
When you pedal a bicycle up a hill, you feel how force must overcome gravity and friction. Using the simple model of constant force, we can relate mass and acceleration with the equation $F = m a$. If the cyclist increases power steadily, the acceleration changes over time, and we can estimate $a$ from velocity data as $a = \\Delta v / \\Delta t$. This example shows how algebra and calculus connect in a real system.

We can also consider air drag, but this basic model already links measurement and prediction.

```text
Given mass m = 70 kg and force F = 140 N, a = F/m = 2 m/s^2.
```
