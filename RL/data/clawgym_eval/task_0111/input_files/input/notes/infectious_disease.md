# Infectious Disease
Q: First-line outpatient treatment for uncomplicated cystitis?
A: Nitrofurantoin for 5 days or TMP-SMX for 3 days (if local resistance low and no contraindications), or fosfomycin single dose.

Q: Meningitis empiric therapy in adults <50?
A: Ceftriaxone + vancomycin; add ampicillin if Listeria risk; plus dexamethasone if pneumococcus suspected.

Q: HIV post-exposure prophylaxis window?
A: Start as soon as possible, ideally within 2 hours and up to 72 hours after exposure; continue for 28 days.
