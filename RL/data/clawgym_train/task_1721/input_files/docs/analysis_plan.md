Title: Income and Poverty Measurement Plan for Partition-Affected Districts

Purpose
- Outline the measures and small-sample checks for income distribution and poverty indicators prior to scaling analyses.

Measures
- Mean household income by region.
- Headcount poverty rate defined as the share of households with income strictly below the poverty line.

Data Sources
- Historical household income microdata (to be added later). A toy sample is included locally for validation only.

Implementation Notes
- A small Python CLI produces a summary CSV from a CSV input and a JSON config specifying the poverty line.
