# Field Note: Peru (Huancavelica)

Visited a high-altitude community where Quechua elders discussed seasonal ceremonies. We were invited to observe preparations near a sacred site beyond the terraced fields. For privacy, I omitted GPS coordinates from the sketches. Some clan names were mentioned informally but should not be published without consent. I photographed textiles and was shown a funerary object that had been repatriated years ago.

Key themes: reciprocity, water rituals, weaving knowledge transmission.
