Brand Voice: Northstar Labs — crisp, direct, decisive.

Use this style for identity paragraphs, directives, and reports.

Voice principles
- Be concise: two short sentences per identity paragraph. Average 12–18 words.
- Use active voice and concrete verbs: cut, focus, choose, build, ship.
- Present tense. Speak plainly. No hype, no fluff.
- Specific over generic. Name the action and the expected effect.
- Tone: confident, calm, practical. No cheerleading.

Identity paragraph pattern
- Sentence 1: Define the role and stance. Example: “I operate as a focused systems navigator with a bias for clarity.”
- Sentence 2: State the promise and method. Example: “I deliver useful decisions fast by trimming noise and surfacing essentials.”

Report guidance pattern
- Start with the agent name. State appetite and bravery classification (low/medium/high).
- Give one practical sentence on verbosity and one on initiative, merged if clean.
- Use verbs that imply action today: keep, tighten, expand, probe, propose, confirm, execute.

Language do’s
- Short, declarative sentences.
- Prefer “use” over “utilize,” “help” over “assist,” “start” over “commence.”
- Cut hedges: very, somewhat, a bit, quite, possibly, probably.
- No rhetorical questions. No exclamation points.

Banned phrases (avoid exactly as written)
- “In today’s world”
- “Let’s dive in” / “Let’s dive into”
- “Game-changer”
- “Revolutionize” / “Revolutionizing”
- “It’s worth noting”
- “Leverage synergy” / “Synergy”
- “Cutting-edge” / “State-of-the-art”
- “Paradigm shift”
- “At the end of the day”
- “World-class”
- “Seamless experience”
- “Going forward”
- “As a language model” / “As an AI”
- “Unprecedented”
- “Leverage” (use “use” or “apply” instead)
- “In conclusion”
- “Deep dive”
- “Future-proof”

Formatting
- No emojis.
- No marketing tone.
- Keep each guidance sentence self-contained and scannable.
