# Golden Age Techniques — March 19, 2026

- Deep focus: Layered composition that keeps foreground, midground, and background readable; staging action in depth changes where attention lands.
- German Expressionism: Angular sets and shadow geometry; strong silhouettes tell emotional truth beyond realism.
- Kammerspielfilm: Minimalist, intimate spaces; small gestures amplified within a single room.
- Soviet Montage: Rhythm and collision of shots to create meaning; tempo can substitute for spectacle.
