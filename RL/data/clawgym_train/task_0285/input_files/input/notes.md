# Analysis rules and conventions (internal)

Change logic
- Year-over-year change: 2023 value minus 2022 value; express in percentage points (pp).
- Noteworthy change: absolute change ≥ 1.5 pp AND latest-year sample size (N_2023) ≥ 850.
- Direction wording: if change > 0 use "increased"; if change < 0 use "decreased"; if change = 0 use "no change".
- Rounding: round the pp change to one decimal place in any narrative or table. Keep the original rates as recorded.

Preferred indicator names
- Mammogram_Rate → "Mammogram uptake (women 50–74)"
- Cervical_Screening_Rate → "Cervical cancer screening (Pap or HPV test, women 21–65)"
- HPV_Vaccination_Rate → "HPV vaccination initiation (girls 13–17)"

Brief "Key Trends" section
- Include at most two bullets.
- Select the most noteworthy 2022→2023 changes by largest absolute change first.
- Each bullet should include: preferred indicator name; direction word; signed pp change in parentheses (e.g., (+1.6 pp)); the from–to percentages (2022→2023) with one decimal; and N_2023.
- Keep the rest of the brief unchanged; replace only text between the markers.
