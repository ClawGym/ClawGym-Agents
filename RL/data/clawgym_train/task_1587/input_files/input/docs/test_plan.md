# Atari 2600 Emulator Test Plan (Community Edition)

This community-maintained plan is used to guide regression checks for common Atari 2600 titles across popular emulators. The focus is on controller behavior, timing stability, and known switching schemes.

## Scope
- Primary audience: community testers
- Platforms: desktop emulators
- Out-of-scope: hardware capture verification
