Welcome to the Library Catalog! This message explains how to search and hopefully find what you need, though there are quite a few different ways to think about search. If you already know the exact book or media item you are interested in, you might try entering a phrase in the search box and then pressing Enter. The system will consider several factors in order to return results, and there are different fields like Title, Author, and Subject that may be looked at. Users sometimes type everything into one big query, but you can also refine later. For example, you could look at facets and filters, but you might also just try multiple attempts. Some people use Subject headings to group similar topics, which can be helpful but is occasionally confusing when combined with other advanced options. If the search is not returning expected results, try simplifying your terms and possibly removing punctuation or extra words, and feel free to ask a librarian for help.
