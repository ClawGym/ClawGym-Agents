Recommended Phrases for De-escalation

  • "I'm here to listen. Let's take a breath together."
  • "What you’re feeling makes sense; we can make a plan."
  • "Would it help to step outside for a moment?"
  • "Let’s focus on one small next step."
  • "We can set boundaries that feel safe for you."
