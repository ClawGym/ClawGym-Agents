Explanation:
A classic model for population growth assumes a limited environment. The logistic equation describes the growth rate as $\\frac{dN}{dt} = r N (1 - N/K)$, where $N$ is population size, $r$ is the intrinsic rate, and $K$ is the carrying capacity. With data from a lab culture, we can fit the model and compare prediction and measurement.

```text
If N=100, r=0.2, K=500, then dN/dt = 0.2*100*(1-100/500) = 16.
```
