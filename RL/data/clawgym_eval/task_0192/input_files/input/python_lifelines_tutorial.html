Cox Proportional Hazards

This tutorial demonstrates fitting a Cox model in Python using lifelines.


from lifelines import CoxPHFitter
cph = CoxPHFitter()
cph.fit(df, duration_col="T", event_col="E")
cph.print_summary()

Assumptions and Diagnostics

  • Proportional hazards: test with Schoenfeld residuals and time-varying coefficients.
  • No strong collinearity among predictors.
  • Log-linearity of numeric effects unless modeled otherwise.
