# 2026-04-15 — Daily Notes

Context: First pass at the onboarding flow for new contributors. Captured checklist steps, owners, and a few decisions.

## Onboarding Checklist

- Create workspace account and set MFA
- Read MEMORY.md and search_tips.md
- Run embedding script to index memory
- Complete “hello world” PR on a low-risk file
- Schedule 15-minute retention check after first week
- Confirm access to project alpha repository

Notes:
- Use explicit headers and clear file paths so hybrid search picks this up by header and filepath.
- Encourage new contributors to skim the “Hybrid Memory Search Tips”.

## Decisions

- Adopt hybrid retrieval with base weights: vector 0.4, keyword 0.25, header 0.1, filepath 0.25
- Use adaptive weighting when keyword overlap is < 0.1
- Turn on temporal routing for explicit dates
- Enable pseudo-relevance feedback when top score < 0.45 and no date is present

Rationale:
- We observed better recall for onboarding queries using adaptive weighting.
- Daily notes benefit from temporal routing to surface the correct date quickly.
