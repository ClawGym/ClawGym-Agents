# Week 3: Innate vs Adaptive Immunity

Learning objectives:
- Explain innate immunity: physical barriers, pattern recognition, phagocytes (macrophages, neutrophils), complement system
- Explain adaptive immunity: B cells, T cells, clonal selection, immunological memory
- Describe antigen presentation and MHC: role of dendritic cells, MHC I vs MHC II
- Discuss cytokines and chemokines in coordinating inflammation
- Apply principles of vaccination to generate immunological memory and neutralizing antibodies
- Distinguish hypersensitivity and tolerance: autoimmunity and allergy (Type I hypersensitivity)
