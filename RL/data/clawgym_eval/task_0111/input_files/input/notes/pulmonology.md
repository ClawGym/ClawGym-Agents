# Pulmonology
Q: Diagnostic criteria for asthma in adults?
A: Episodic symptoms with variable airflow limitation, improved FEV1 by >=12% and 200 mL after bronchodilator or positive methacholine challenge.

Q: First-line treatment for COPD exacerbation?
A: Short-acting bronchodilators (albuterol/ipratropium), systemic glucocorticoids, antibiotics if indicated, supplemental oxygen.
